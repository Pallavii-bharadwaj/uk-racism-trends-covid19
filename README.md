# 🇬🇧 UK Racism Trends Analysis During COVID‑19

### 🎯 Objective
Analyse UK racism incident trends during the COVID‑19 pandemic to explore temporal and geographic patterns and apply a responsible data governance framework.

### 🧠 Background & Motivation
The COVID‑19 period saw increased reports of racially motivated incidents in the UK.  
This project applies ethical and responsible data-handling practices to curate, validate, and analyse these datasets — ensuring that conclusions are transparent, accountable, and data-driven.

### 🧰 Tools & Technologies
Python | Pandas | NumPy | Matplotlib | Seaborn | Scikit-learn | Data Governance Frameworks

### 📈 Methodology
1. **Data Governance** — examined source credibility, provenance, and consistency  
2. **Data Cleaning** — handled missing values, normalized categories, ensured metadata accuracy  
3. **EDA** — visualized trends by time, region, and type of incident  
4. **Statistical Modelling** — explored correlations and seasonality patterns  
5. **Visualization** — produced heatmaps, time-series plots, and dashboards for policymakers

### 🔍 Results & Insights
- Identified notable spikes in racism-related incidents following certain pandemic milestones  
- Clear correlation between regional demographics and reported incidents  
- Demonstrated the value of consistent reporting and transparent data sharing

### 📘 Learning & Reflection
- Reinforced importance of **ethical AI and data governance**
- Improved **EDA and visualization** workflows for social datasets  
- Practised communicating sensitive findings responsibly
