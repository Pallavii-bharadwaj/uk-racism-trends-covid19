# 🇬🇧 UK Racism Trends Analysis During COVID-19

## 🎯 Objective
Analyse UK racism incident trends during the COVID-19 pandemic to explore temporal and geographic patterns and apply a responsible data governance framework.

## 🧠 Background & Motivation
The COVID-19 period saw increased reports of racially motivated incidents in the UK. This project applies ethical and responsible data-handling practices to curate, validate, and analyse these datasets — ensuring that conclusions are transparent, accountable, and data-driven.

## 🧰 Tools & Technologies
**R** | **R Markdown** | **tidyverse** (dplyr, tidyr) | **ggplot2** | Statistical Modelling | Data Governance Frameworks

## 📈 Methodology
1. **Data Governance** — examined source credibility, provenance, and consistency
2. **Data Cleaning** — handled missing values, normalised categories, ensured metadata accuracy
3. **EDA** — visualised trends by time, region, and type of incident
4. **Statistical Modelling** — explored correlations and seasonality patterns
5. **Visualisation** — produced heatmaps, time-series plots, and visual summaries for policy-relevant communication

## 🔍 Results & Insights
- Identified notable spikes in racism-related incidents following certain pandemic milestones
- Clear correlation between regional demographics and reported incidents
- Demonstrated the value of consistent reporting and transparent data sharing

## 📘 Learning & Reflection
- Reinforced the importance of ethical data handling and governance frameworks
- Improved EDA and visualisation workflows for sensitive social datasets
- Practised communicating findings responsibly on a high-stakes topic

## 🗂️ Repository Structure
- `analysis.Rmd` → R Markdown file containing data cleaning, EDA, and statistical modelling
- `evens_teaching_dataset.tab` → raw tab-separated dataset used for analysis

Open `analysis.Rmd` in RStudio and Knit to HTML to reproduce the full analysis and visualisations.
